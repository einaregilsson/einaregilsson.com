Yet Another Random Quote, with editing

Posted: 2007-07-30 Last updated: 2012-04-18

UPDATE 01.05.2012: No idea if this still works, it probably doesn't.

On my old site I used to have a random quote feature. When I moved to WordPress I missed it so I found a plugin that displayed random quotes. It was called Yet Another Random Quote (yarq for short) and lives at http://openmonday.org/2006/07/31/yet-another-random-quote/. It's a good plugin and works well, but it could only add and delete quotes, not edit existing ones. So I changed it a bit, added editing functionality and moved the quote text fields above the quote list, which I thought was better when you had a lot of quotes. After I'd changed it I emailed the author twice to ask him if he wanted to use my changes in his next version but never heard back from him. So IÂ´m gonna distribute my version here:
